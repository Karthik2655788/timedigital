Digital Clock & Countdown Timer

Project Overview

This project is a simple and interactive web application built using HTML, CSS, and JavaScript. It displays a live digital clock that updates every second and includes a countdown timer showing the remaining time until New Year 2027. The application also provides a Dark Mode and Light Mode toggle for a better user experience.

⸻

Features

1. Digital Clock

* Displays the current time in HH:MM:SS format.
* Updates automatically every second.
* Uses JavaScript’s Date() object.

2. Countdown Timer

* Counts down to January 1, 2027.
* Displays:
    * Days
    * Hours
    * Minutes
    * Seconds
* Updates every second.

3. Theme Toggle

* Switch between Dark Mode and Light Mode.
* Enhances user experience and accessibility.
* Implemented using JavaScript event handling and CSS classes.

4. Responsive Design

* Adapts to different screen sizes.
* Works on desktops, tablets, and mobile devices.

How to Run

1. Create a file named index.html.
2. Copy and paste the project code into the file.
3. Save the file.
4. Open index.html in any web browser.
5. The digital clock and countdown timer will start automatically.
